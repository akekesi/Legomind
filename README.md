Legomind_README
Attila Kekesi
23.05.2020

Legomind is a board game/mobile app idea based on Mastermind.
The game is a combination of classic Mastermind and Lego-Minifigures.
Goal: Guess a randomly assembled Lego Character
Players: 1, 2
